Moss认识：

Moss在其Read Me中介绍了其项目：将 Monad 上复杂的 DApp/协议交互抽象为 Agent 可调用的统一能力，并由系统而不是 Agent 负责组装正确的交易。概括来讲，这是一个让ai可以完成链上操作打包，最后再交由用户进行签名授权的ai相关项目，目的是让ai再不接触用户钱包隐私的同时准备链上交易。其可以用于让用户直接调用ai寻找能够完成用户意图的协议，再读取参数和风险说明，生成未签名交易计划，并在真实链上状态中模拟检查资产变化。例如用户需要进行货币兑换，可以直接让ai寻找合约并生成交易和检查，最终由用户决定是否签名。
