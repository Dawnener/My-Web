Web3Monad Testnet学习记录

今天我完成了 Web3 Summer Internship 的第一个合约部署任务。

今天完成了什么

今天我完成了 Web3 Summer Internship 的第一个合约部署任务。

我使用 Remix 编译了一个最小 Solidity 合约，并将它部署到了 Monad Testnet。部署完成后，我保存了合约地址和部署交易 hash，也完成了至少一次合约交互。

我学到了什么

这次任务让我第一次理解了智能合约部署的大致流程。

合约部署到测试网之后，不会因为我关闭 Remix、钱包或浏览器页面而消失。只要部署成功，它就已经记录在链上，之后可以通过区块浏览器继续查询。

遇到的问题

  • 什么是交易 hash

当前理解

合约地址可以理解为这个合约在区块链上的“位置”。

交易 hash 可以理解为某一次操作的“记录编号”。

read function 是读取链上数据，通常不需要花 gas。

write function 是修改链上数据，通常需要钱包确认，并产生交易记录。
