Moss AI Agent Exploration

Project 02
AI AgentGitHubOpen Source

通过阅读开源代码与参与文档协作，理解 AI Agent 如何连接 Web3 能力。

项目目标

理解 Moss 如何把复杂的链上 DApp 与协议交互整理为 Agent 可调用的能力，并学习开源项目的 Issue、Pull Request 和审核协作流程。

完成内容

  • 阅读 Moss 的 README、Docs、Example、Issue 和 Pull Request；
  • 围绕初学者工作流整理 examples/simple-flow/README.md；
  • 根据 Issue 的需求完成文档内容，并提交 Pull Request；
  • 根据维护者反馈调整内容，使说明更贴近可运行代码。

成果与证据

查看 Moss 初学者流程 README

阅读 README 与 PR 学习记录

阅读 Issue 协作学习记录

这次实践带来的理解

开源协作不只是提交代码。先确认问题、阅读上下文、让文档准确贴合项目结构，再根据反馈修改，都是一次贡献能真正被项目接纳的重要部分。
